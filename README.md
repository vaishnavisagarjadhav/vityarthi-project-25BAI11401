This is a cute, clean, and fully functional money-tracking program I built to help myself (and anyone else!) stop spending too much.  
No databases, no internet, no complicated stuff ‚Äî just pure Python that runs anywhere!

---

## ‚ú® Features

| Feature                            | Description                                                  |
|------------------------------------|--------------------------------------------------------------|
| Add Income                         | Record money you received (salary, allowance, gifts, etc.)  |
| Add Expenses                       | Track everything you spend                                   |
| Live Balance                       | Always shows your current total                              |
| Transaction History                | See every income and expense in order                        |
| Simple Menu                        | Only 4 options ‚Äî super easy to use                           |
| Input Validation (basic)           | Warns if you type something wrong                            |
| Decimal Support                    | Handles dollars and cents correctly (e.g., 12.99)            |
